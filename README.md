# 🛒 Retail Sales Analysis (SQL Project)

📌 Project Overview
This project analyzes retail sales data using SQL to extract business insights.

🛠 Tools Used
- SQL (PostgreSQL)
- Data Cleaning
- Aggregations
- Group By
- Subqueries
- KPI Analysis

📊 Key Analysis Performed
- Total sales calculation
- Top 5 customers by revenue
- Sales by shift (Morning, Afternoon, Evening)
- Revenue by product category
- Customer behavior analysis

🎯 Objective
To demonstrate practical SQL skills in querying, cleaning, and analyzing transactional data.
